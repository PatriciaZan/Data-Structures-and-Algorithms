Tree

A DS Tree -> fundamental gierarchial data structure, a collection of elements(data), called nodes, conected by edges(links).

Full Tree -> each node either has no children(leaf node) or exactly two children.

Perfect Tree -> Full + balance(nodes have same depth/distance from the root) having no gaps at any level.

Complete Tree -> all levels except possibly the last one are completed filled with nodes.
Left leaning Last Level -> filled with nodes from the left side as much as possible. Might be empty on the right.

BST(Binary Search Tree)

Organize data in a sorted way. Efficiently search, add, or remove items.

  • Go LEFT if value < than the current branch
  • Go RIGHT if value > than the current branch
